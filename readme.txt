Lorem Ipsum (Demo Text or Dummy Text) generator for your WordPress editor.

== Description ==

This plugin adds a new button into your WordPress page/post editor which allows you to insert Lorem Ipsum (demo text) quickly.
It's useful for creating demo content.

Check the video
[youtube https://www.youtube.com/watch?v=m4PIEJednNo]

== Installation ==

Search and install under plugins menu on your wp-admin.
